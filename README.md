## 关于视频源的导入导出

视频源的导入导出是通过剪贴板完成的，格式为json字符串。

视频源导入步骤：

1. 导出已有的视频源到剪贴板
2. 复制到任何地方
3. 根据这个json格式，完善你自己的视频源
4. 导入视频源，切换播放即可

视频源查找直接百度【资源站】
